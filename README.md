# Python Hello World

这是一个简单的Python Hello World项目，用于测试IDE和GitHub仓库的对接。

## 项目结构

```
python-hello-world/
├── hello_world.py    # 主程序文件
├── .gitignore        # Git忽略文件
└── README.md         # 项目说明文档
```

## 如何运行

1. 确保您的电脑已安装Python 3.x
2. 运行以下命令：
```bash
python hello_world.py
```

Hello, World!
